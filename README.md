## JogoRpgPoo - Programação Orientada Objeto

### Pré requisitos
- VS Code
- Extensões C#
- SDK da microsoft
- Crie uma pasta nos arquivos 
- Inicie o projeto no terminal com Dotnet new console
- Se estiver fora do VS code, digite .  code e enter, te levará ate à IDE
